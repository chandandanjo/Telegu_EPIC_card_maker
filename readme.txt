Requirements :- Pillow, pyqrcode.

Keep the 'image.jpg','sign.jpg','fn.jpeg','hn.jpeg' and all the fonts in the same folder as the 'e_card.py' file itself.

If you want to change the font type and size, open the 'e_card.py' file in a notepad and goto line 37,38,39 for bold, normal and telugu font respectively and change the font name to your font name and keep it in the same folder.

Open to improvements.
